A mini search engine built to understand how real-world search systems work beyond SEO concepts.

While these topics explained how websites are ranked, they did not explain how a search engine itself actually works internally.
This gap sparked my curiosity.
To deeply understand the end-to-end working of a search engine, I decided to build a mini search engine from scratch, implementing the core components that real-world search systems use.

## Features
- Web crawling using sitemaps
- Keyword-based search (BM25)
- Semantic search using embeddings
- Hybrid ranking (keyword + semantic)
- Result highlighting
- Strict handling of invalid queries
